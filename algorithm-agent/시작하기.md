# AlgoChiri — 시작 가이드

## 폴더 구조

```
분석기 폴더/
├── algorithm-agent/
│   ├── algochiri.py          ← 에이전트 메인 코드
│   ├── .env              ← API 키 (직접 만들어야 함)
│   ├── requirements.txt  ← 필요한 라이브러리
│   └── 시작하기.md
├── 인스타 분석/
│   └── insta-analyzer.html
├── 카카오 분석/
│   └── kakao-analyzer.html
└── hub.html
```

---

## STEP 1 — API 키 발급

1. https://platform.claude.com 접속
2. 계정 만들기 (구글 계정으로 가능)
3. 로그인 후 왼쪽 메뉴 → **API Keys** 클릭
4. **Create Key** 버튼 클릭
5. 이름 입력 (예: longchiri-agent) → 생성
6. `sk-ant-...` 로 시작하는 키 복사 (한 번만 보여요! 꼭 복사하세요)

---

## STEP 2 — 파이썬 설치 확인

터미널(Mac) 또는 명령 프롬프트(Windows) 열고:

```bash
python3 --version
```

Python 3.10 이상이면 OK. 없으면 https://python.org 에서 설치.

---

## STEP 3 — 라이브러리 설치

`algorithm-agent` 폴더로 이동 후:

```bash
pip install -r requirements.txt
```

---

## STEP 4 — API 키 설정

`.env.example` 파일을 복사해서 `.env` 로 이름 변경 후 키 입력:

```
ANTHROPIC_API_KEY=sk-ant-여기에_실제_키_입력
```

---

## STEP 5 — 실행!

```bash
python3 algochiri.py
```

에이전트가 자동으로:
- 알고리즘 최신 정보 검색
- 분석기 파일 업데이트 필요 여부 판단
- 필요하면 파일 수정 + hub.html 로그 업데이트
- 결과 리포트 출력

---

## 자동 실행 설정 (선택)

### Mac — 매주 월요일 오전 9시 자동 실행

터미널에서:

```bash
crontab -e
```

아래 내용 추가 (경로는 본인 것으로 수정):

```
0 9 * * 1 cd /Users/본인이름/분석기폴더/algorithm-agent && python3 algochiri.py >> agent.log 2>&1
```

### Windows — 작업 스케줄러

1. 시작 → 작업 스케줄러 검색
2. 기본 작업 만들기
3. 트리거: 매주 월요일
4. 동작: 프로그램 시작 → `python3 algochiri.py`
