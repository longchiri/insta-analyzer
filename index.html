📥 Instagram 데이터 내보내는 방법
①instagram.com 접속 → 우측 상단 프로필 아이콘 클릭
②좌측 메뉴 설정(톱니 아이콘) 클릭
③개인정보 및 보안 → 데이터 다운로드 클릭
④정보 요청 탭 → 다운로드할 계정 선택
⑤형식: JSON 선택 → 날짜 범위 설정 → 요청 제출 클릭
⑥며칠 후 이메일로 링크 발송 → 링크 클릭 → ZIP 다운로드
⑦다운로드한 .zip 파일을 위 업로드 영역에 드래그 또는 클릭으로 업로드
💡 PC 웹은 계정 센터를 거치지 않고 바로 설정 → 데이터 다운로드 메뉴로 접근할 수 있어요.
①인스타그램 앱 실행 → 우측 하단 프로필 탭 이동
②우측 상단 ≡ 메뉴 탭 → 설정 및 개인정보
③계정 센터 탭 → 내 정보 및 권한
④정보 다운로드 → 일부 정보 다운로드
⑤다운로드할 계정 선택 → 정보 유형 선택(전체 또는 원하는 항목)
⑥형식 JSON, 날짜 범위 설정 → 파일 만들기
⑦며칠 후 이메일 도착 → 링크 클릭 → ZIP 다운로드 → 여기에 업로드
💡 Android는 파일 앱에서 다운로드 폴더를 확인하거나, 이메일 앱에서 직접 링크를 열어 다운로드하세요.
